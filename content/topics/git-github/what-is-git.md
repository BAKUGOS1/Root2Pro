# What is Git?

Git is a version control system. It helps you save checkpoints of your work so you can track changes, go back safely and collaborate with others.

## Simple example

Think of Git like save points in a game. Every commit is a checkpoint.

## Why it matters

Developers use Git to work safely on projects without losing history.

## Common mistake

Beginners often think GitHub and Git are the same. Git is the tool. GitHub is a platform that hosts Git repositories.
