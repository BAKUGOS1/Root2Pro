# What HTML Does

HTML gives structure to a web page. It tells the browser what each part of the page means.

## Simple idea

HTML is like the skeleton of a web page. CSS gives style and JavaScript adds behavior.

```html
<h1>My Profile</h1>
<p>I am learning web development.</p>
```

## Recall

What does HTML provide: style, behavior or structure?
