# Headings & Paragraphs

Headings and paragraphs are the basic elements used to present readable text layouts on the web.

## Headings (`<h1>` to `<h6>`)

HTML provides six levels of headings. `<h1>` is the most important (highest level) and `<h6>` is the least important.
```html
<h1>Main Title (Use only one per page)</h1>
<h2>Section Title</h2>
<h3>Sub-section Title</h3>
```

## Paragraphs (`<p>`)

The `<p>` tag is used to wrap blocks of text. Browsers automatically add a small margin above and below paragraphs.
```html
<p>This is a paragraph containing text on my website.</p>
```

## Inline Formatting

You can style text inside headings and paragraphs:
* **Strong Importance:** Use `<strong>` to make text **bold**.
* **Emphasis:** Use `<em>` to make text *italic*.
```html
<p>Make sure you <strong>always</strong> structure your HTML correctly.</p>
```
