# College Event Page

Your goal is to build an informational webpage for an upcoming college event, like a hackathon or a cultural fest.

## Requirements
- Use semantic HTML (`<header>`, `<main>`, `<section>`, `<footer>`).
- Create a schedule using an HTML `<table>`.
- Add a registration form using `<form>`, `<input>`, and `<button>`.
- Link to an external map or sponsor sites.
