# Contributing to Root2Pro

Thank you for helping improve Root2Pro.

## Contribution rules

- Keep explanations simple and human-friendly.
- Explain why a topic matters before syntax.
- Add examples, practice and recall.
- Include common mistakes where useful.
- Do not copy external content.
- Prefer small pull requests.

## Pull request checklist

- [ ] The content is original.
- [ ] The explanation is beginner-friendly.
- [ ] Code examples are easy to verify.
- [ ] Graph IDs and content IDs match.
- [ ] `npm run validate` passes.
